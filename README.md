# shiftworkOA
Files for "Associations of Sleep and Shift Work With Osteoarthritis Risk"

Programs used to generate results for:
Yanik EL, Bridgeman A, Herzog ED, Pham V, Evanoff BA, Guilak F. “Associations of Sleep and Shift Work with Osteoarthritis Risk.” Arthritis Care Res. 2026 Jan. Epub ahead of print.

Table12345 = SAS code for generating all result tables in the manuscript
Figures_Rcode = R code for generating all figures in the manuscript

Datasets derived from the UK Biobank (http://www.ukbiobank.ac.uk/) under Application 27034.
